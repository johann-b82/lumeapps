# Getting Started

Welcome to the KPI Dashboard documentation.

## Uploading Data

You can upload CSV, TXT, or Excel files via the upload page.

### Supported Formats

- CSV (comma-separated)
- TXT (tab-separated)
- Excel (.xlsx)
